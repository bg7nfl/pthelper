所有被管理主机需要安装agent

  • 

    Centos 安装 需要使用自定义yum源 

    • yum install python36-setuptools python36-django python36-osquery python36-requests python36-simplejson python36-djangorestframework python36-gunicorn python36-greenlet python36-gevent python36-supervisor osquery
    • yum install opsadmin-agent
  • 

    Debian 安装 需要使用自定义apt源 

    • apt-get install python3-setuptools python3-django python3-osquery python3-requests python3-simplejson python3-djangorestframework python3-gunicorn python3-greenlet python3-gevent python3-supervisor osquery
    • apt-get install opsadmin-agent
  • 

    Agent启动方式

    • systemctl start supervisord
    • 启动后使用端口20080,访问http://ip:20080检查是否安装成功
